# 🌟 Featured Projects

## 🌐 Epic NCR Realty

🔗 http://www.epicncrrealty.com

### 🏠 Description

A full-stack real estate web platform developed to manage and showcase residential and commercial property listings across the NCR region. The platform provides a smooth and responsive experience for buyers and agents with advanced property search, inquiry management, and lead generation features.

### ⚙️ Key Features

* Property listings with images, pricing, location, and amenities
* Advanced filtering by property type, location, and budget
* User authentication system
* Inquiry & contact forms
* Admin dashboard for property management
* Fully responsive UI
* Secure backend and database handling

### 🛠️ Tech Used

HTML, CSS, JavaScript, Bootstrap, Python, Django, SQLite/MySQL
